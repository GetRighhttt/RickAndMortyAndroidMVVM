# RickAndMortyAndroidMVVM
RIckAndMortyMvvm app just to practice paging and caching data with a recycler view. I will be adding in majority of the other jetpack components as well.

#### Here is a list of all the features used in this application:

- Design Patterns
- Paging3
- LoadStateListener
- SearchView
- Swipe Mechanism
- Retrofit2
- API
- Coroutines
- Dagger-Hilt
- MVVM
- LiveData
- Coroutines
- Sealed Classes
- Intents: PutExtra
- ROOM
- Navigation Drawer
- Glide
- Recycler View
- Animations
- View Binding
- Material3
- Material Dialogs
- Material Toolbar
- Snackbar
- Splash Screen
